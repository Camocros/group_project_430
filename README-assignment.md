# group_project_430
## Design Theme

Our project will use a clean, modern, and responsive design.

### Color Scheme

- Primary Color: Blue `#2563EB`
- Secondary Color: Green `#22C55E`
- Background Color: Light Gray `#F9FAFB`
- Text Color: Dark Gray `#111827`
- Accent Color: Orange `#F97316`

### Typography

- Main Font: Arial or Inter
- Headings: Bold and large for clear structure
- Body Text: Simple and readable

### General Layout

The application will have a simple layout with:

- A navigation bar at the top
- A main content area
- Cards to organize information
- Buttons with rounded corners
- Responsive design for mobile and desktop

### Component Ideas

- Header / Navbar
- Home Page
- Login Form
- Dashboard Cards
- Footer
- Buttons
- Form Inputs
